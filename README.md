# spotify-billboard-ml-project
A prediction Machine Learning Model of what the most ideal season to release Music is Based on Historical Data.

# Music Data Analysis & Machine Learning

## Overview
This project analyzes music datasets from Spotify and Billboard to understand patterns in song popularity and chart performance.

## Methods
- Data cleaning and normalization using regex
- Merging multiple datasets
- Feature engineering (cyclical encoding of months)
- Random Forest model

## Results
- Evaluated using MAE and R²
- Model shows relationships between release timing, genre, and chart performance

## Tools
Python, Pandas, Scikit-learn, Matplotlib
